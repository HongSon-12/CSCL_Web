# Docs

Thư mục tài liệu được chia theo mục đích:

| Thư mục | Nội dung |
|---|---|
| `plan/` | Tài liệu chính để triển khai CSCL Web: BRD, PRD, FRD, implementation plan, kiến trúc |
| `data-catalog/` | Catalog bảng, cột và measure dùng để chuyển dashboard/Power BI sang web + Python |
| `legacy-agent-ai/` | Tài liệu thiết kế hệ Agent-AI hiện có, dùng làm bối cảnh khi tích hợp vào portal |

Khi triển khai, ưu tiên tài liệu trong `plan/`. Tài liệu `legacy-agent-ai/` chỉ dùng để tránh phá vỡ logic Agent-AI cũ.
